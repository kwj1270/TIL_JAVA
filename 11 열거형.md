# enum 이란?     
> Enumerated type : 열거형 타입      
     
enum이란 enumerated type 의 줄임말로          
**멤버**라 불리는 명명된 값의 **집합**을 이루는 자료형이다.      
즉, 열거형에 사용될 수 있는 특정한 값들을 정의해서 해당 값들만 사용할 수 있게 한다.                
참고로, 열거자 이름들은 일반적으로 해당 언어의 **상수** 역할을 하는 식별자이다.
